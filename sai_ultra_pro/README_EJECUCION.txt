# Manual de Ejecución SAI ULTRA PRO

1. Instala dependencias necesarias (ver requisitos en cada script).
2. Configura tus API keys en config/config.json.
3. Ejecuta main.py desde VS Code:
   
   ```
   python sai_ultra_pro/main.py
   ```
4. Revisa alertas en Telegram y registros en Google Sheets.

No uses scalping ni VPS. Solo ejecuta desde tu PC personal.
